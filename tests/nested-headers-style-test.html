🎨 NestedHeaders Style 功能测试指南

📋 测试 1：基础背景色和文字色

配置代码：

nestedHeaders: [
    [
        {
            label: "基本信息",
            colspan: 2,
            style: {
                backgroundColor: "#4472C4",
                color: "#FFFFFF"
            }
        },
        {
            label: "工作信息",
            colspan: 4,
            style: {
                backgroundColor: "#70AD47",
                color: "#FFFFFF"
            }
        },
    ],
    ["姓名", "年龄", "城市", "部门", "薪酬", "入职日期"]
]

预期效果：

✓ 第一层"基本信息"显示为蓝色背景 + 白色文字
✓ 第一层"工作信息"显示为绿色背景 + 白色文字
✓ 第二层使用默认样式（无特殊颜色）
  • "基本信息"表头背景色为 #4472C4（深蓝）
  • "基本信息"表头文字为白色
  • "工作信息"表头背景色为 #70AD47（绿色）
  • "工作信息"表头文字为白色
  • 第二层表头保持默认样式不变

📊 测试 4：多层级差异化样式

场景：创建一个 3 层嵌套表头，每层使用不同的视觉强度

nestedHeaders: [
    // 第 1 层：强样式（大色块 + 白字 + 粗体）
    [{ label: "分类A", colspan: 2, style: {
        backgroundColor: "#5B9BD5", color: "#FFF",
        fontWeight: "bold", fontSize: "15px", textAlign: "center"
    }}],

    // 第 2 层：中等样式（浅色背景 + 半粗体）
    [{ label: "子类1", colspan: 1, style: {
        backgroundColor: "#BDD7EE", fontWeight: "600"
    }}, "子类2"],

    // 第 3 层：无特殊样式
    ["字段1", "字段2"]
]
层次感预期：
┌──────────────────┐
│     分类A    │ ← 深蓝底、白字、15px、粗体、居中
├────────┬─────────┤
│ 子类1  │  子类2  │ ← 浅蓝底、半粗体
├────────┼─────────┤
│ 字段1  │  字段2  │ ← 默认样式
└────────┴─────────┘
  • 第一层视觉权重最高（最深/最大/最突出）
  • 第二层视觉权重中等（浅色/正常大小）
  • 第三层最简洁（与普通列表头一致）
  • 整体呈现清晰的层次结构
