📊 嵌套表头导出示例（无数据场景）

⚠️ 问题背景（v1.0.12 及之前版本）：
  • 当有嵌套表头但没有数据时，导出的 Excel 文件是空的（0 字节）
  • 用户期望：即使没有数据，嵌套表头也应该被正确导出
  • 影响场景：模板下载、空表结构导出等
✅ 修复后行为（v1.0.13+）：
  • ✅ 嵌套表头会被完整导出到 Excel 文件中
  • ✅ 合并单元格、样式等都会保留
  • ✅ 用户可以基于此模板填写数据

🎯 功能演示

💡 核心功能特性

📝 模板下载

提供带格式的 Excel 模板供用户填写，包含完整的嵌套表头结构和样式。

📊 表结构导出

仅导出表格结构（表头），不含实际数据，适用于数据迁移和架构设计。

🔄 数据迁移

先导出结构，再导入到其他系统，保持数据格式的一致性。

📑 报表框架

创建标准化的报表格式，确保所有输出文档的统一性和专业性。

💻 代码示例

// 创建工作簿（无数据，仅配置嵌套表头）
const container = document.getElementById('app');
const workbook = new Workbook(container, {
    startRows: 0,  // 无数据行
    startCols: 6,
});

// 配置嵌套表头
workbook.activeSheet.nestedHeaders = [
    [
        { label: "基本信息", colspan: 2 },
        { label: "工作信息", colspan: 4 },
    ],
    [
        "姓名",
        "年龄",
        { label: "城市" },
        { label: "部门" },
        { label: "薪酬", colspan: 2 },
    ],
    ["Name", "Age", "City", "Dept", "Salary", "Hire Date"],
];

// 导出（即使没有数据，嵌套表头也会被导出）
await workbook.downloadFile('xlsx', {
    filename: 'template-with-headers.xlsx',
    nestedHeaders: true,      // 关键选项：启用嵌套表头导出
    cellStyles: true,         // 可选：包含样式
});
