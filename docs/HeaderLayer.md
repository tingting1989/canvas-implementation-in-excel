# HeaderLayer — 表头层

## 概述

`HeaderLayer` 是渲染引擎中负责渲染 **Excel 行号头和列标头** 的图层。它位于视口最顶部的固定区域，提供行列标识功能。

## 类结构

```
BaseLayer
  └── HeaderLayer (zIndex: 50)
        └── headerRenderer: HeaderRenderer  // 表头渲染委托
```

## 可视化布局

```
┌──────────┬──────┬──────┬──────┬──────┐
│  (空白)   │  A   │  B   │  C   │  D   │  ← 列标头
├──────────┼──────┼──────┼──────┼──────┤
│    1     │      │      │      │      │
├──────────┼──────┼──────┼──────┼──────┤
│    2     │      │      │      │      │  ← 行号头 + 数据区
├──────────┼──────┼──────┼──────┼──────┤
│    3     │      │      │      │      │
└──────────┴──────┴──────┴──────┴──────┘
```

## 核心职责

| 职责 | 说明 |
|------|------|
| 列标头渲染 | 绘制 A, B, C... 列标识，支持冻结列时的分段渲染 |
| 行号头渲染 | 绘制 1, 2, 3... 行标识，支持冻结行时的分段渲染 |
| 选区高亮 | 高亮选中范围的行列头背景色 |
| 拖拽指示 | 显示列宽/行高拖拽时的调整线 |

## 图层关系

```
zIndex 70: UILayer         ← 调试信息、冻结线
zIndex 50: HeaderLayer     ← 【本层】行号头 + 列标头
zIndex 40: FrozenLayer   ← 冻结区域数据
zIndex 10: TileLayer       ← 主数据区域
```

## 状态监听

| 键 | 触发场景 |
|----|----------|
| `scroll` | 滚动位置变化，表头需重新定位 |
| `frozen` | 冻结配置变化，表头需调整冻结部分 |
| `viewport` | 视口尺寸变化，表头需重新计算布局 |
| `selection` | 选区变化，表头需高亮选中行列 |

## 渲染流程

```
render()
  └→ headerRenderer.render()
       ├→ 渲染列标头背景
       ├→ 绘制列分隔线和文字 (A, B, C...)
       ├→ 渲染行号头背景
       ├→ 绘制行分隔线和文字 (1, 2, 3...)
       ├→ 高亮选中列的列标头
       ├→ 高亮选中行的行号头
       └→ 渲染拖拽调整指示线（如有）
```

## 使用示例

```js
const headerLayer = new HeaderLayer();

// 绑定到 RenderEngine
engine.addLayer(headerLayer);
```
